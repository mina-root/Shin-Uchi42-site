真打42 Air (Gen2)

ユーザーガイド / 使い方の説明

Warning

使用上の注意

液体をかけたり、高温になる場所に放置したり、強い力や衝撃を与えたりしないでください。接触不良やバッテリーの発火を起こす可能性があります。

直射日光に長期間当てないでください。高温を避ける意味でも、筐体を傷めないという意味でも。

日本国外で電源を入れないでください。違法電波になる可能性があるので。

Start

使い始める

左右それぞれで、内側の側面にあるの電源スイッチを上に切り替えて電源を入れます。

PCのBluetooth設定画面で "Shin_Uchi42 Air" が表示されたら接続します。入力方式の指定が出た場合は ANSI（US配列） を選択してください。

左側が親機で、右側が子機（ZMKの初期設定）になっています。子機は親機に接続して動作し、単体では動作しません。

Default Keymap

デフォルトキーマップ

初期状態では以下のキーマップが設定されています（購入時期によって異なる場合があります）。中央の文字が通常入力、下部の文字は長押しで入力されます。

大西配列を採用しています。QWERTY配列ではありませんのでご注意ください。

キーに印字がないので、慣れないうちはキーマップを確認（画面上に表示しておくか、印刷などをしておく）しながら使うことをおすすめします。大丈夫、使っているうちに慣れますよ。

英数キー長押しで記号数字レイヤーが、かなキー長押しで操作系レイヤーが有効になります。

Shiftは、どちらの手でも人差し指か中指のホームポジションのキーを長押しすることで入力されます。

ミニボタンは、左手側を左から①②、右手側を左から③④としたとき、以下の機能が割り当てられています。

  • ①：数字記号レイヤーの数字と一部記号をテンキー入力か通常の入力か切り替える
  • ②：Ctrl+Sが押される こまめに保存しよう！
  • ③：Bluetoothプロファイルを切り替えるためのBTレイヤーに切り替える
  • ④：BTプロファイルを次の番号に切り替える（4番の次は0番に戻ります）
  • ①+②：NumLockが押される 数字記号レイヤーはテンキー入力モードがデフォルトなので、NumLockはオンの状態で使うことを前提としています
  • ③+④：ブートローダーモードに入る（キーボードファームウェアの書き換えに使えます）

TABは左手側の中央行の右2つのキーを同時押しすることで入力できます。2つのキーのちょうど間の部分を人差し指で押すのがおすすめです。

Bluetoothプロファイルを切り替えるためのレイヤーは、右手側の左のミニボタンを押している間有効化されます。

Charging

充電について

USB Type-Cケーブルを使用し、電源スイッチをオンにした状態で充電してください。充電中は端子横上面の穴のうち右側から見えるLEDが緑色に点灯します。

長期間使用しない場合は、必ず電源をオフにして保管してください。また、バッテリーの劣化を防ぐため、半年に一度は充電してください。

子機は通常使用において親機よりも数倍バッテリーが持つため、親機ほど頻繁に充電する必要はありません。

Remap

キーマップを書き換える

もちろん、キーマップは書き換え可能です。詳細な手順は以下のリポジトリを参照してください。

当然、QWERTY配列にすることもできます。が、カラムスタッガードキーボードでQWERTY配列はあまり相性が良くないのでおすすめはしません。大西配列以外にも、日本語・英語それぞれにQWERTYより最適化された配列があるので、自分だけの最強の配列を見つけてくださいね。

Shin-Uchi42Air_ZMK-config

リセットスイッチの位置

USB-C端子付近の上面にある2つの穴のうち、左側の穴の奥にリセットスイッチがあります。細い棒で押してください。デフォルトキーマップだと右側のミニボタン2つの同時押しでもリセットできます。

Support

困ったときは

Bluetooth接続ができない、入力が不安定などの症状が出る場合は以下を確認してください。

  • 充電切れの可能性は？ → 充電してくださいね。
  • 他ホストに接続中かも？ → BT (番号)で空いているプロファイルに変更するか、BT Clear / BT Clear Allで登録されているプロファイルを削除してください。
  • 記号が違うよ？ → PC側の入力方式が日本語配列になっているならUS配列に変更してください。あるいはキーマップ設定を間違えているかも。
  • キーが無反応なんだけど？ → キーマップ設定を確認して、問題なければハード的な不良の可能性。ミナモト@mina_VRChへ連絡してくださいね。
